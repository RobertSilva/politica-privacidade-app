Política de Privacidade — Cálculo Escolar

Última atualização: 17 de julho de 2025

1. Coleta de Informações

Este aplicativo não coleta dados pessoais como nome, e-mail, localização, fotos ou qualquer outro tipo de dado identificável.

As informações fornecidas pelo usuário, como disciplinas e notas escolares, são armazenadas localmente no dispositivo por meio de banco de dados SQLite.

2. Uso de Dados

Os dados inseridos são usados apenas para:

  • Realizar cálculos de média escolar;
  • Organizar disciplinas e avaliações;
  • Exibir dados em listas interativas.

Os dados não são enviados nem compartilhados com terceiros.

3. Permissões

O app solicita a permissão de acesso à Internet apenas para exibir anúncios. Nenhum dado pessoal é transmitido por essa conexão.

4. Anúncios

O aplicativo exibe anúncios de terceiros, como o Google AdMob. Esses anúncios podem utilizar identificadores não pessoais para personalização. Para saber mais, consulte a Política de Privacidade da Google.

5. Público-Alvo

Este app é recomendado para estudantes com idade igual ou superior a 13 anos, especialmente do 6º ano do ensino fundamental até o 3º ano do ensino médio.

6. Segurança dos Dados

Todos os dados inseridos ficam armazenados apenas no dispositivo. O app não realiza backups automáticos. O usuário pode apagar os dados diretamente no app ou desinstalá-lo para removê-los completamente.

7. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página com frequência. Mudanças significativas serão notificadas no aplicativo.
